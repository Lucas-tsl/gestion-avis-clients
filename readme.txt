# ⭐ Gestion Avis WooCommerce

> Contrôlez la visibilité des avis produits WooCommerce et intégrez Google Avis Clients facilement – sans toucher au code.

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)
![Licence](https://img.shields.io/badge/licence-GPLv2-blue.svg)
![Compatible WooCommerce](https://img.shields.io/badge/compatible-WooCommerce-brightgreen.svg)

---

## 🧩 Fonctionnalités principales

- ✅ Limite les avis produits affichés à 3 (avec bouton “Voir plus”)
- ⭐ Ajoute un script Google Avis Clients sur la page “Merci pour votre commande”
- 🎛 Menu admin dédié avec deux sections de configuration simples

---

## 🛠️ Installation

1. Télécharge ou clone ce dépôt dans le dossier `wp-content/plugins/`
2. Active le plugin via l’interface WordPress
3. Va dans **Gestion Avis** (menu admin) pour :
   - Activer/désactiver la limitation des avis
   - Saisir ton **ID Marchand Google** (Merchant Center)
