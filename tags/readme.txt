Un plugin pour woocommerce pour gérer les avis produits et injecter Google Avis Clients. Il permet une configuration simple >
== Description ==

**Gestion Avis Clients** vous permet de :

- Limiter les avis produits (nombre ou conditions)
- Injecter Google Avis Clients avec vos identifiants
- Afficher une page d'administration centralisée pour gérer les retours
- Offrir une meilleure expérience post-achat

Idéal pour améliorer la confiance des clients et maîtriser votre image.


== Installation ==

1. Téléversez le dossier du plugin dans `wp-content/plugins`
2. Activez-le depuis le menu Extensions dans WordPress
3. Configurez-le depuis l’onglet “Gestion Avis” dans l’administration

== Frequently Asked Questions ==

= Est-ce compatible avec tous les thèmes WooCommerce ? =
Oui, le plugin utilise les hooks standards de WooCommerce.

= Puis-je désactiver la partie Google Avis Clients ? =
Oui, toutes les options sont désactivables dans la page d’administration.
